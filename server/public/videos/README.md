# Demo Videos Folder

Place your demo video files here (MP4, WebM, etc.)

## How to Use:

1. Copy your video file to this folder:
   ```
   server/public/videos/my-demo-video.mp4
   ```

2. In Admin Dashboard â†’ Demo Videos, use this URL:
   ```
   http://localhost:5000/videos/my-demo-video.mp4
   ```

3. For production, update the URL to your domain:
   ```
   https://yourdomain.com/videos/my-demo-video.mp4
   ```

## Recommended Video Formats:
- **MP4** (H.264 codec) - Best compatibility
- **WebM** - Good for modern browsers
- **Resolution:** 720p or 1080p
- **File size:** Keep under 50MB for better loading
